Welcome to Scratchpad!

the minimalist document editor


Scratchpad is a new kind of document editor. The goals of scratchpad are to

  • help you focus on writing
  • make it really easy and fast to take notes
  • make it easy to extend the editor with extra features
  • work on any device
  • easily add interactive features to documents

You can also add quotes to documents. They look best if they are at least two lines long in length.

Scratchpad uses material-framework for its design, and has text editing features based on grande.js. You can add bold text, italics, and links. type a * character or a number and a period on a new line to start a list. Select text to format it or turn it into a heading.

Interactive components
